# 05 Transform Objects

在`threejs`中有四个属性可以旋转

- position
- scale
- rotation
- quaternion

## position

`position`的 x、y、z 分别对应 x 轴、y 轴、z 轴

```js
cube.position.x = 0.5;
cube.position.y = -0.9;
cube.position.z = 0.1;
```

`cube.position.set(0.5, -0.9, 0.1)`这一行与上面表达相同意思
